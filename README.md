# Min Browser Userscripts

Koleksi userscript untuk [Min Browser](https://minbrowser.org/) yang dirancang untuk meningkatkan pengalaman browsing, produktivitas, dan estetika.

## Daftar Script

### 1. Dark Mode (`dark-mode.js`)
Memberikan fitur Dark Mode pada semua situs web yang dikunjungi.
- **Shortcut:** Tekan `Alt + Shift + D` untuk menyalakan atau mematikan.
- **Fitur:** Menggunakan filter cerdas untuk menginversi warna tanpa merusak kualitas gambar dan video. Status (on/off) disimpan secara otomatis per situs.

### 2. YouTube Optimized (`Youtube.js`)
Optimasi total untuk pengalaman menonton YouTube yang lebih bersih.
- **Fitur:**
    - **Ad-Blocker:** Melewati iklan video secara otomatis dan menghapus banner iklan.
    - **No Shorts:** Menghilangkan rak YouTube Shorts dari beranda dan sidebar agar lebih fokus.
    - **Anti-Popup:** Melewati deteksi ad-blocker dan menutup popup "Try Premium".
    - **Ringan:** Dirancang untuk performa tinggi tanpa membebani browser.

## Cara Instalasi di Min Browser

1. Buka **Min Browser**.
2. Masuk ke menu **Edit** > **Preferences** (atau **Min** > **Preferences** di macOS).
3. Gulir ke bawah hingga menemukan bagian **Userscripts**.
4. Klik tombol untuk membuka folder userscripts di komputer Anda.
5. Salin file `.js` dari repository ini ke dalam folder tersebut.
6. Restart Min Browser atau refresh tab yang sedang terbuka.
